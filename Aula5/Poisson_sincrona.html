Modelo de Regressão de Poisson

O modelo de regressão de poisson costuma ser aplicado em estudos nos quais a variável de interesse é contagem, ou seja, uma variável quantitativa discreta.

O Modelo de poisson faz parte da classe dos Modelos Lineares Generalizados, casos nos quais a distribuição dos erros não necessariamente possui uma distribuição Normal.

O objetivo dos Modelos Lineares Generalizados é obter para uma variável resposta que não é quantitativa contínua uma relação linear nos parâmetros das variáveis preditoras, ou seja, um preditor linear \(\eta = \beta_0 +\beta_1X_{1i}\), a partir de uma função de ligação que relaciona a variável resposta com as variáveis preditoras.

A função de ligação utilizada no Modelo de Regressão de Poisson é a função \(log\).

\[log(\lambda_i)= \beta_0 + \sum_{j=1}^k \beta_jX_{ij}\]

# Criando uma função para estimar o cálculo de razão de taxas

glm.RR <- function(GLM.RESULT, digits = 2) {

    if (GLM.RESULT$family$family == "binomial") {
        LABEL <- "OR"
    } else if (GLM.RESULT$family$family == "poisson") {
        LABEL <- "RR"
    } else {
        stop("Not logistic or Poisson model")
    }

    COEF <- stats::coef(GLM.RESULT)
    CONFINT <- stats::confint(GLM.RESULT)
    TABLE <- cbind(coef = COEF, CONFINT)
    TABLE.EXP <- round(exp(TABLE), digits)

    colnames(TABLE.EXP)[1] <- LABEL

    TABLE.EXP
}

Podemos interpretar os modelos bivariados cujos preditores foram significativas dessa forma:

m1) A cada aumento de um morador por domicílio, o número médio de casos aumenta em 3 vezes \((RP = 3,1)\)

m2) Temos uma redução na média de casos na ordem de \(0,89\) ou de \([(1-0,89) \times 100 = 11\%\) para cada aumento percentual de domicílios com coleta de lixo regular.

m3) Temos uma redução na média de casos na ordem de \(0,98\) ou de \([(1-0,98) \times 100 = 2\%\) para cada aumento percentual de percentual de domicílios com esgotamento sanitário adequado.

m4) Temos uma redução na média de casos na ordem de \(0,88\) ou de \([(1-0,88) \times 100 = 12\%\) para cada aumento percentual de alfabetizados.

m5) Temos uma redução na média de casos na ordem de \(0,006\) ou de \([(1-0,006) \times 100 = 99\%\) para cada aumento de unidade do índice de desemvolvimento social.

m6) Temos uma redução na média de casos na ordem de \(0,99\) ou de \([(1-0,99) \times 100 = 1\%\) para cada aumento de um real na renda médiada população.

m7) A cada aumento percentual de pessoal sem renda declarada, o número médio de casos aumenta em \(1,028\) vezes.

m8) Esse resultado indica os bairros com um percentual maior que \(30\%\) de pessoas sem renda declarada, apresentam uma média de casos de leptospirose \(1,472\) vezes maior que os bairros com um percentual menor que \(30\%\) de pessoas sem renda declarada.

O que é a Deviance ou função desvio ?

  • A deviance é uma métrica que está relacionada ao erro, quanto menor desvio significa que o modelo se ajusta melhor aos dados.

  • Quanto maior o desvio, pior o modelo se ajusta aos dados em comparação ao melhor modelo (modelo saturado).

  • Deviance é uma estatística de qualidade de ajuste para um modelo frequentemente usado para testes de hipóteses.

Temos então:

\[\text{Null Deviance} = 2 \times [ll(\text{Saturated Model}) - ll(\text{Null Model})]\]

\[\text{Residual Deviance} = 2 \times [ll(\text{Saturated Model}) - ll(\text{Proposed Model})]\]

Deviance ou estatística D

\[D = \text{Deviance}_{residual} \sim \chi^2_{1-\alpha; n-(p+1)}\]

Estatística de Pearson

\[\chi^2 = \sum_{i=1}^n \dfrac{(y_i - \hat{\lambda}_i)^2}{\hat{\lambda}_i}\]

\[\chi^2 \sim \chi^2_{1-\alpha; n-(p+1)}\]

[1] 782.1701
[1] 8.188935e-86
  • Como a hipótese nula \((H_0)\) é que nosso modelo está especificado corretamente e como o \(p-valor < 0,001\), temos fortes evidências de dizer que o modelo não se ajusta muito bem aos nossos dados.

Estatística \(D^2\)

  • Calcula a quantidade da deviance explicada pelo modelo, ou seja, é a proporção do desvio na variável resposta explicada pelo modelo.

\[D^2 = 1 - \dfrac{\text{Deviance}_{residual}}{\text{Deviance}_{nula}}\]

[1] 0.2525752
  • O modelo está conseguindo explicar \(D^2 =\) 25.26 \(\%\) da ocorrência dos casos de leptospirose.

Pseudo \(R^2\) de McFadden

  • A interpretação é semelhante ao do \(R^2\) do modelo de regressão linear.

\[R^2_{MF} = 1 - \dfrac{LL \text{ (modelo proposto)}}{LL \text{ (modelo nulo)}}\]

[1] 0.423211
  • O modelo consegue explicar somente 42.32 \(\%\) da variabilidade da ocorrência dos casos de leptospirose.

Medidas de Influência

Uma alternativa para verificação de medidas influentes está implementada no pacote car.

  • O primeiro gráfico apresenta os valores da distância de Cook para cada observação. A distância de Cook é uma medida de diferença das estimativas dos parâmetros do modelo ao considerar e ao desconsiderar uma particular observação no ajuste.

  • O segundo gráfico mostra os resíduos studentizados. Um modelo bem ajustado apresenta estes resíduos dispersos aleatóriamente em torno de 0, entre \(-3\) e \(3\) desvios.

  • O terceiro gráfico mostra os valores dos pontos de alavanca (\(h_{ii}\)). Valores elevados são considerados potencialmente influentes. Os valores desses pontos de alavanca estão entre 0 e 1.

Com base nesses 3 gráficos, podemos observar que não há indicativos fortes de outliers ou observações influentes principalmente para o modelo binomial negativo.
